Sistema de batalha RPG desenvolvido em Java com foco em Programação Orientada a Objetos.

## Funcionalidades
- Combate por turnos
- Troca de personagens
- Sistema de ataque e defesa
- Inimigo com IA básica
- Gerenciamento de vida e estados

## Conceitos aplicados
- Classes e objetos
- Herança
- Polimorfismo
- Encapsulamento
- Abstração
- Condicionais e estruturas de repetição
- Organização em pacotes

## Tecnologias
- Java
